# 마이크로 서비스

마이크로 서비스 구현

<br/>

 ## 개발 동기

마이크로 서비스의 작동 방식을 알아보기 위한 프로젝트

<br/>

## 주요 기능

- Micro Service의 설정 파일을 중앙에서 관리하는 기능 (Config Server)
- Micro Service의 작동 여부를 확인하는 기능 (EurekaServer)
- Micro Service의 GateWay (Gateway)
- Micro Service가 다른 Micro Service를 호출하는 기능

<br/>

## 개발 환경

- `Spring boot`
- `Spring Cloud`
- `IntelliJ`
